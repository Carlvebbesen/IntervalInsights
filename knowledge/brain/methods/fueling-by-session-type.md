# Fueling by Session Type

> A practical map of *what each running session type is* and *how much to fuel it* — the applied
> form of `[[fuel-for-the-work-required]]`. Session definitions follow `[[src-maurten-fuel-guides]]`;
> the brain's house focus is the threshold work of the `[[norwegian-method]]`.

## How it works — the per-session table

| Session type | What it is | Fuel **during** |
|---|---|---|
| **Sprints** | 5–15 s near-max reps, full recovery | none |
| **Easy / recovery** | low-intensity, slower than marathon pace | water (sip in heat); no carbs needed |
| **VO₂ max** | 2–4 min reps at 3–10k pace (e.g. `[[4x4-norwegian]]`, `[[1000m-reps]]`) | none if < ~60–75 min; carbs **before**; sips of drink mix between sets if long/depleted |
| **Lactate tolerance** | 150–600 m at 800–1500 m race pace | none during; fuel + `[[bicarbonate]]` **before** |
| **Threshold** | controlled reps at 2–3 mmol/L (`[[10min-threshold-reps]]`, `[[lactate-threshold]]`) | carbs **before**; **1–2 sips of drink mix between reps** to hold quality |
| **`[[double-threshold]]` day** | two same-day threshold sessions | as above each session; **refuel hard between AM & PM** (race season) |
| **Long run** | > ~90 min, slower than marathon pace (`[[progressive-long-run]]`) | **30–60 g/h** (≈ 1 gel/h) |
| **Marathon-pace tempo** | continuous MP block (`[[marathon-pace-tempo]]`) | carbs before; if long, **30–60 g/h** — rehearse race fueling |
| **Racing — HM / marathon** | — | **60–90 g/h** (marathon ≈ 1 gel/6 km) — needs `[[multiple-transportable-carbohydrates]]` |
| **Racing — ultra** | — | **30–60 g/h**, plan solids + `[[hydration]]` |

## Timing (around any fueled session)

- **Pre, 1–4 h before:** a carbohydrate meal/feed; add `[[bicarbonate]]` before high-intensity work
  (lactate tolerance, threshold, hill reps).
- **Warm-up (10–30 min):** a gel for sessions that need fueling.
- **Post:** refuel to restock glycogen — most urgent **before a second session** (double-threshold)
  or the next day's quality.

> [!tip] Timing pages: `[[pre-training-nutrition]]` → `[[during-training-nutrition]]` →
> `[[post-training-nutrition]]`; the daily base is `[[meal-pattern]]`.

## When to use it

- Whenever prescribing or planning a session — pair the workout with its fueling row so the
  intended `[[lactate-threshold]]` / `[[vo2max]]` stimulus is actually achievable.

## Evidence & caveats

- Targets and session definitions are from `[[src-maurten-fuel-guides]]` (prescriptive). The
  ~60 → 90 g/h jump for racing rests on `[[multiple-transportable-carbohydrates]]` (Jeukendrup
  consensus, not the source).
- The **~60–75 min** "do you need to fuel" line is a guideline — individualise, and **train the gut**
  before relying on high race-day intakes.
- Always plan `[[hydration]]` alongside, especially in heat.
