# Functional Threshold Power (FTP)

> The highest power (watts) a cyclist can hold in a roughly steady state for ~1 hour — the
> power-meter proxy for `[[lactate-threshold]]`, and the anchor that all power-based load metrics
> are scaled to.

## Key facts

- **The reference unit for `[[training-stress-score]]`:** one hour **at** FTP = **TSS 100**, and
  **Intensity Factor (IF)** is effort expressed relative to FTP (1.0 = at threshold).
- It is the practical, field-testable stand-in for the second `[[lactate-threshold|lactate
  threshold]]` (LT2 / MLSS), and it sets the boundaries of power-based `[[training-zones]]`.
- Conceptually the cycling analogue of threshold pace in running — the brain's default
  intensity control is still **measured lactate**, not power (`[[control-intensity-by-measurement]]`);
  FTP is the metric the TSS/PMC family is built on.
