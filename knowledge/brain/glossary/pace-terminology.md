# Pace & Recovery Terminology

> The personal (Norwegian) shorthand used across the ingested running sessions. Hub page so sessions stay readable.

## Pace references

| Term | Norwegian | Meaning | ~Zone (`[[training-zones]]`) |
|------|-----------|---------|------------------------------|
| **mf / mp** | maratonfart | marathon-race pace | Z3–Z4 |
| **hmf / hp** | halvmaratonfart | half-marathon-race pace | Z4 (~`[[lactate-threshold]]`) |
| **10k** | 10k-fart | 10 km race pace | Z5 |
| **400-fart** | — | ~400 m race pace (very fast) | Z5+ |

## Recovery / structure terms

| Term | Meaning |
|------|---------|
| **flyt / flytpause** | "float" recovery — controlled, *faster than jog* (not fully easy) |
| **ståpause** | standing rest |
| **joggepause** | easy jogging recovery |
| **ap** | aktiv pause (active pause — easy jog) |
| **stigningsløp** | strides |
| **langtur** | long run |
