# Norwegian 4×4

> Four 4-minute efforts near maximal aerobic intensity — a classic, well-studied `[[vo2max]]` builder.

```interval
discipline: run
reps: 4
work: 4 min
work_intensity: Z5
recovery: 3 min
recovery_intensity: Z1
sets: 1
target: VO2max
notes: ~90-95% HRmax on work intervals; classic Norwegian protocol
```

## Purpose

- Maximizes time spent near `[[vo2max]]`, the key stimulus for raising aerobic ceiling.
- A VO₂max sharpening tool / weekly "X element" — used **sparingly** in the default
  `[[norwegian-method]]` (where controlled threshold, not VO₂max work, is the base staple); it is
  the staple high-intensity 20% only under the alternative `[[polarized-training]]`.

## Execution notes

- Warm up thoroughly (10–15 min) before the first rep.
- Target the **top of Z4 / Z5** in `[[training-zones]]` — hard but evenly paced; don't
  sprint the first rep and fade.
- Recoveries are easy (Z1) but active. 1–2 sessions/week max; pair with easy days
  (`[[recovery-and-adaptation]]`).
