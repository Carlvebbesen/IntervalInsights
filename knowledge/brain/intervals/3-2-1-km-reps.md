# 3-2-1 km Reps (descending)

> A descending 3 / 2 / 1 km set with 2 min active pause; repeat 2–3× and add a set for marathon training.

> [!warning] Demanding session — high volume at intensity. Assumes a well-developed `[[aerobic-base]]` and consistent, high-mileage training. Not for beginners.

```interval
discipline: run
reps: 3 km, 2 km, 1 km (descending), × 2-3 sets
work: 3 km / 2 km / 1 km
work_intensity: starts marathon/half-marathon pace, gets faster as reps shorten
recovery: 2 min active pause (ap)
recovery_intensity: easy jog
sets: 2-3
target: lactate-threshold / vo2max
notes: add an extra set for marathon-specific volume
```

## Purpose

- Progressive descending reps spanning threshold to 10k pace; trains pacing and
  `[[lactate-threshold]]`/`[[vo2max]]`. Open at MP/HMP (`[[pace-terminology]]`).
