# 1600-1200-800-600-400 (descending track)

> A descending-ladder track session with **shrinking recovery** — the Kenyan / Moroccan
> race-pace workout. You want lactate to **climb steadily**, not spike.

> [!warning] Demanding session — race-pace track intensity. Assumes a well-developed `[[aerobic-base]]` and sharpening phase.

```interval
discipline: run
reps: 1 set of (1600, 1200, 800, 600, 400 m)
work: 1600 / 1200 / 800 / 600 / 400 m
work_intensity: race pace; lactate 4-8 mmol/L (>10 in racing season)
recovery: reduces down the ladder, ~1 min -> 30 s
recovery_intensity: easy
sets: 1
target: race-specific / vo2max
notes: Kenyan track + Moroccan (coach Kada) race-pace session; a steady lactate rise = controlled absorption, not "dying" (cf. a 1200/800 ladder climbing 5.8 -> 9.3 mmol/L)
```

## Purpose

- Race-specific speed and lactate tolerance. The shrinking recovery forces a **gradual,
  controlled lactate rise** across the set — the Kenyan marker of a session done right.

## Execution notes

- Don't blow up the first long rep. The goal is a smooth climb in lactate, finishing fast but
  controlled — *not* a spike on rep one. Sits in the [[kenyan-training]] track-session band.
