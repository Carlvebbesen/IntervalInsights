# 15-25 × 400 m

> Fifteen to twenty-five 400 m reps with a short 40–45 s rest — a high-rep VO₂max/speed session.

> [!warning] Demanding session — high volume at intensity. Assumes a well-developed `[[aerobic-base]]` and consistent, high-mileage training. Not for beginners.

```interval
discipline: run
reps: 15-25
work: 400 m
work_intensity: fast (10k pace → faster; VO2max/speed)
recovery: 40-45 s
recovery_intensity: easy
sets: 1
target: vo2max / speed
notes: short rest keeps the aerobic demand high across many reps
```

## Purpose

- Develops `[[vo2max]]` and leg speed/economy. In a marathon/half-marathon block these add
  speed reserve so race pace feels easier. Pace refs in `[[pace-terminology]]`.

## Origin & threshold variant

- **25 × 400 m every Wednesday** was Joe Newton's staple at York H.S. (on 160 km/wk) and became
  a standard session in the `[[norwegian-method]]` (`[[src-bakken-norwegian-model-revisited]]`).
- **Threshold variant (double-threshold evening):** the same distance run *much* easier —
  **20–25 × 400 m at only ~3.0 mmol/L** with a **15–30 s** float — is Bakken's Tuesday-evening
  short-threshold session, the fast-but-controlled half of a `[[double-threshold]]` day. Same
  reps, very different intensity: know which one you're doing (`[[control-intensity-by-measurement]]`).

## Execution notes

- Build the rep count across the cycle; 25 × 400 is a large dose. For the **VO₂max** version hold
  the short 40–45 s rest; for the **threshold** version keep lactate ≤ ~3.0 and don't race it.
