# 28 km Progressive Long Run

> A 28 km long run that steps down in pace every 4 km, finishing well under marathon pace.

> [!warning] Demanding session — high volume at intensity. Assumes a well-developed `[[aerobic-base]]` and consistent, high-mileage training. Not for beginners.

## Structure (pace ladder, min/km)

| Segment | Pace |
|---------|------|
| 0–8 km | 5:00 |
| 8–12 km | 4:50 |
| 12–16 km | 4:40 |
| 16–20 km | 4:30 |
| 20–24 km | 4:20 |
| 24–28 km | 4:10 |

## Targets

- Builds `[[aerobic-base]]` and marathon durability, finishing near/below marathon pace
  (`[[pace-terminology]]`) on tired legs — a strong marathon-specific stimulus.

## Fueling

> [!tip] Fuel the work
> A run this long needs carbohydrate — aim **30–60 g/h** (≈ 1 gel/h) and drink to thirst
> (`[[carbohydrate-fueling]]`, `[[hydration]]`). It's the ideal place to **rehearse race-day
> fueling**; full per-session guidance in `[[fueling-by-session-type]]`.

## Execution notes

- Start genuinely easy; the progression should feel controlled until the final segments.
- Per the note: keep the day *after* a long run very easy (recovery — `[[recovery-and-adaptation]]`).
